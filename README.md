# 個人財務管理系統 (FinanceManager) - v2.0 (GUI 升級版)

這是一個基於 Java 開發的個人財務管理工具。在 v2.0 版本中，我們從傳統的命令列介面全面進化為現代化的圖形使用者介面 (GUI)，提供更直觀的記帳與分析體驗。

## 🚀 版本更新：v2.0 (GUI 與視覺化)
- **現代化介面**：引入 **FlatLaf** 皮膚，支援現代化的深色模式視覺效果。
- **視覺化分析**：新增「消費趨勢分析」功能，以圖形化長條圖呈現支出比例。
- **進階搜尋**：支援透過下拉選單選擇類別進行「關鍵字搜尋」或執行「大額支出篩選」。
- **預算監控**：主介面即時顯示預算進度，並提供自動警示顏色變換。

## 🛠️ 環境需求
- **JDK 17** 或更高版本。
- **外部依賴庫 (Jar 檔案)**：
  - `gson-2.13.1.jar`：用於 JSON 資料處理。
  - `flatlaf-3.5.1.jar`：用於 GUI 皮膚美化。

## 📦 如何在 VS Code 執行
1. 開啟專案資料夾。
2. 在左側 **Java Projects** 檢視窗中，找到 **Referenced Libraries**。
3. 點擊 **+** 號，將 `gson-2.13.1.jar` 與 `flatlaf-3.5.1.jar` 同時加入。
4. 打開 `FinanceGUI.java` 並點擊右上角的 **Run** 按鈕啟動。

## 📂 檔案結構
- `FinanceGUI.java`: 系統主入口與視窗操作邏輯。
- `CategoryChart.java`: 獨立的繪圖元件，負責產生消費分析圖表。
- `FinanceService.java`: 財務運算與資料處理邏輯。
- `FileHandler.java`: 負責 JSON 檔案的讀取與寫入。
- `Transaction.java`: 交易紀錄模型。
- `FinanceData.java`: 定義 JSON 存儲結構。

## 📝 功能介紹
1. **快速記帳**：點擊「新增紀錄」可透過下拉選單快速選擇歷史類別，減少打字時間。
2. **預算警示**：當月支出超過預算 80% 時，狀態列文字會變更為橘色並顯示警告。
3. **動態圖表**：即時計算各分類佔比並以視覺化長條圖呈現。
